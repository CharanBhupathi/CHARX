# Portfolio Website

A stunning portfolio website with a full-screen background image, handwriting-style navigation, and visit counter.

## Features

- **Full-screen background image** - Add your own image as `background.jpg`
- **Handwriting-style navigation** - Links to About, Projects, Habits, and Other pages
- **Visit counter** - Tracks and displays the number of visits with a smooth hover effect
- **Background music** - Plays automatically when the website opens (add your music file as `music.mp3`)

## Setup Instructions

1. **Add your background image:**
   - Place your image in the project folder
   - Name it `background.jpg` (or update the path in `styles.css`)

2. **Add your music file:**
   - Place your music file in the project folder
   - Name it `music.mp3` (or update the path in `index.html`)
   - Supported formats: MP3, OGG

3. **Open the website:**
   - Simply open `index.html` in your web browser
   - Or use a local server for better experience

## File Structure

```
CHARX/
├── index.html          # Main page
├── about.html          # About page
├── projects.html       # Projects page
├── habits.html         # Habits page
├── other.html          # Other page
├── styles.css          # All styling
├── script.js           # Visit counter and music functionality
├── background.jpg      # Your background image (add this)
└── music.mp3           # Your music file (add this)
```

## Customization

- Edit `styles.css` to change colors, fonts, and layout
- Modify `script.js` to adjust visit counter behavior or music settings
- Update the content in each HTML file to add your personal information

Enjoy your stunning portfolio!
